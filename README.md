# Digital Escape: Museum

## Overview
**Digital Escape: Museum** is a **2D top-down escape room puzzle game** developed in **Unity using C#**.  
The player explores a museum environment and solves logic-based puzzles to unlock new rooms and eventually reach the final exit.

The project focuses on **environmental exploration, puzzle-solving mechanics, and structured progression between rooms**, simulating the experience of a digital escape room.

## Gameplay
In *Digital Escape: Museum*, the player controls the character **Lila**, who navigates through a series of museum rooms.  
Each room contains clues and interactive objects that help the player solve puzzles.

The main gameplay loop is:

**Explore → Find clues → Solve puzzle → Unlock next room → Repeat**

Progression is controlled through **interactive puzzles and locked turnstiles**, ensuring that each challenge must be completed before advancing.

At the end of the game, the player reaches the final room and triggers the completion sequence.
